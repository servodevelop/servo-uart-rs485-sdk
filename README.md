该仓库包括华馨京科技 uart&rs485 系列舵机的SDK例程。

通信协议以及更多资料请参考： [Fashion Star Wiki](https://wiki.fashionrobo.com/)

| STM32F407    | STM32F407 SDK（HAL库&标准库）                              |
| cpp          | C++ SDK                                                    |
| CSharp       | C# SDK                                                     |
| python       | python SDK                                                 |

| arduino      | arduino SDK                                                |
